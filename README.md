susebootstrap v0.1
=============

Install an openSUSE base system into a subdirectory of another, already installed system.


openSUSE Versions
------------

With this scipt you can install following openSuSE Version:
* 12.3  i586 x86_64
* 12.2  i586 x86_64
* 12.1  i586 x86_64

Tested on openSUSE 12.3 only.
